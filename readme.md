DwvExplorer
===========

[Orthanc](http://www.orthanc-server.com/) plugin for the [dwv](https://github.com/ivmartel/dwv/wiki) (DICOM Web Viewer).
This plugin allows to preview DICOM series and instances stored in Orthanc using DWV. It is based on the default OrthancExplorer and just changes the target of the preview buttons. More details on the [wiki](https://github.com/ivmartel/DwvExplorer/wiki).
